## Quick Start

### Step1
```
git clone git@github.com:sunqunyan/.vim.git ~/.vim
```

### Step2
```
git clone git@github.com:VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

### Step3
```
ln -s ~/.vim/vimrc ~/.vimrc
```

### Step4
```
vim .vim/vimrc
```

### Step5
execute vim command:  
```
:PluginInstall
```
